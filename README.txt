# Tresloukadu - Entrega Motoboy para Magento

Esta é a primeira versão do módulo de frete (motoboy) para a loja virtual 
Magento. A loja está funcional e mostra o preço da entrega do motoboy com 
base no bairro selecionado. Nesta primeira versão a lista com bairros e 
preços tem que ser informada no próprio código do módulo, mas em futuras
versões tudo será armazenado no banco de dados.

http://www.tresloukadu.com.br/category/php/magento/


## Requirements

1. PHP 5.2.x
2. Magento 1.7.x

## Usage

Os arquivos devem ser upados de acordo com a estrutura de diretórios do 
módulo. Após será necessário acessar o admin em Configurações -> Métodos 
de envio -> Tresloukadu - Motoboy e ativar o módulo. Caso você tiver a 
opção de compilação ativida no magento você deverá recompilar tudo antes 
de acessar o módulo em configurações.

## Installation

Todos os arquivos devem ser upados através do FTP para a pasta do magento
respeitando a estrutura de diretórios do módulo.
